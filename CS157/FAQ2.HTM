Problem 2:

Are we allowed to use Existential and Universal quantifiers?

Yes.

On part c, do you mean all kings play chess on all finely grained sand? Or that there are some kings who play chess on some finely grained sand?

It should be interpreted this way: if any king comes across a chess board on any patch of finely grained sand, as long as the weather is nice, he'll start playing chess.

Problem 3:

Do you want just an interpretation or a variable assignment as well?

Just an interpretation is fine.

Problem 4:

Are we allowed to use existentials in our sentences?

No. We have been a bit unclear about what we are calling relational logic, so we've been lenient with the use of existentials, but for this particular problem, it is really only interesting if we stick to allowing variables, but without quantifiers. In other words, your sentences should not contain 'A' or 'E'. We assume universal quantification for all variables. So don't use existentials.

Problem 5:

Do you mean "let phi and psi be sentences in relational logic" instead of "phi and t"?

Yes, that was a mistake. It has been fixed.

What interpretations satisfy the empty set of sentences?

All interpretations satisfy the empty set of sentences. No interpretations satisfy a set of unsatisfiable sentences.
